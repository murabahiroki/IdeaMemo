# ■ 1. 金属は燃え尽きるのか？

はい。
**金属は大気圏突入の摩擦熱で普通に蒸発します。**
実際に人工衛星の部品（鉄・アルミ・ステンレス）は、
ほとんど燃え尽きて地上に到達しません。

金は融点1,064℃ですが、
大気圏突入時は外皮温度 1,700〜2,500℃ に達するので完全に溶けます。

しかし問題はここから。

---

# ■ 2. “溶ける”と “消える” は別の問題

金は単体で蒸発しにくい金属。
つまり、

**溶ける → 飛び散る → 微細粒子になって散乱する**

という挙動になりやすい。

* 一部は蒸発（ただし金は蒸発しにくい）
* 一部はミリ〜サブミリサイズの球状粒子になって散らばる
* 地表に届く量は微量

この“球状の金属粒子”は、隕石片（スフェルール）と似た挙動をとる。

---

# ■ 3. 質量が大きければ「一部だけ」地表に落ち得る

もし **“衛星サイズの金塊”**（狂ってるがフィクションならOK）なら：

* 本体の99％ → 空中で蒸発・散逸
* 中心部の“溶け残り”が核（コア）となって
* バラバラになりながらも地表へ落ちる可能性がある

例：
スペースシャトル Columbia の残骸は
アルミ構造なのに 2,000℃ で燃えた後でも
一部は地表まで到達しています。

---

# ■ 4. 現実的なイメージ

■ 表面が蒸発して飛び散る
■ 粒子化した金の“金煙”が何百キロも広がる
■ 地表には “砂金レベルの粉” が落ちるだけ**

地表にたどり着くのは1〜10g以下レベル。

**宝探しどころではない“散乱”**。
