# 第4の事件
番外地アウトロー襲撃事件
江戸川と旧江戸川にはさまれた河原を密かにねぐらにしていた。ハングレが５人殺された。
いずれもトクリュウの幹部と海外組織とのつながりを担っていたと思われ、本庁は監視を続けつつ、警視庁と千葉県警のどちらが担当するかもめていた時代もトクリュウ対策班の誕生で速やかに編成されるようになった。
全員の死亡と火災により現場の状態は異常に荒れていた。

# 対黒幕
　１年後、遠山は高裁の判事に昇格していた。仙台高等裁判所が赴任地である。

　遠山は判決後の展開に一つの仮説を思いつく
　　「証拠固めをしている最中の闇バイト従事者が殺されている・・・・まさか口封じ？」
　　そして4つの事件の共通点に気が付く、
　　　犯人は犯罪者というよりはむしろ正義感
　　　なぜか、闇バイトの人間の行動をつかめた
　　　妙な高揚感で正義の天誅を果たしたと思っている。
　　　被害者は全員警察が証拠固めをしている人間だった。
　　　第１から第３までは関東での事件だったのに、トクリュウ強化後発生した第4の事件は地方・・・これは何を意味するのか？

　　　まさか月見里が後ろで意図を引いていた？いや、そんな様子はなかった。第一真相を暴くメリットがない。

　　　遠山は改めて事件関係資料を読み直す。
　　　奇妙なことに気が付いた。もう一人すべての事件にかかわっていた人間がいたのだ。すべての現場でとばっちりを受けていた第二の被害者ともいうべき人間がいた。その人間こそすべての犯人を陰から操っていた。
　　　ゆがんだ正義感が起こした正義の秘密組織のリーダーっぽい言動。
　　　しかし裏では、すべての犯罪のマネーロンダリングにかかわり、証拠が残らないよう適度に犯人グループのリセットを行いながら、闇のお金の入りと出で儲けていた。

## 章立て＋あらすじ（起承転結）】
### 物語の始まり
* **導入（第1章）**：風祭の赴任、いたずら、生徒との距離感
### 物語の中盤
* **発展（第2～3章）**：不審火・万引きなど事件発生、共通点の兆し
* **転換（第4～5章）**：カツアゲ事件、繋がる点と黒幕の輪郭
# 物語の結末：
* **結末（第6章）**：最終対決・風祭の成長・解決と余韻
