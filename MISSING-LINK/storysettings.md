# 作品基本設定：『MISSING LINK』

【前提】
1926年12月3日のアガサ・クリスティ（36歳）の「11日間の失踪事件」は、実は現代（2020年代）へのタイムスリップ期間だったという解釈のメタ・ミステリー。

【五部構成】

* 第一部：白鳥館（現代で目覚めたテレサの適応、第一の事件）
* 第二部：変わらぬ悪意（女子大生の相棒と日本旅行、合気道入門など。現代特有の法で裁けない悪意の収集）
* 第三部：空白（宿の書棚でミステリーや文学を読む。ミステリーでは普通の読者の様に、推理を楽しんでいたが、アガサクリスティの作品だけ、答え合わせの様に読んでいる。アガサクリスティって誰？Wikipediaで調べるとアクロイド殺し以降に失踪し断筆と書かれていた。つらつらと読みロザリンドの名前を見たときに己の正体に気が付く）
* 第四部：レッサーケイマン（組み立てた現代版『そして誰もいなくなった』のプロットを宿の夫婦と刑事の前で披露。刑事の正体を暴き、格闘の末に刑事は自滅、テレサは雷と共に消失）
* 第五部：11Days After（1926年12月14日のホテルへ帰還。手元には兵隊人形の絵。現代には解答編の手紙が残る）

【厳守すべきロジック・固有設定】

1. 格闘技：30代のアガサは実在のサフラジツ（エディス・ガラッドの柔術）を認知。現代で合気道を猛特訓。身体能力は高いが超人ではなく基本に忠実。
2. 悪役の刑事：白鳥館主人の友人でサイコパス。過剰捜査で不起訴を連発する。テレサの「心理・しぐさの観察」によって正体を見破られ、格闘時の自滅（テーブル崩落）で敗北。
3. ホテル滞在：現代の旅先での行動（食器、ドレス、散歩）が、1926年のホテルでの静養中の目撃証言と無意識にリンクする。
4. 相棒：テレサの行動力に振り回される現代の女子大生。秋（10〜11月）の休業期間に日本を案内する。
5. テレサの仕事：白鳥館で上流階級の優雅なアテンド（メニュー手書き、海外客対応）をこなす。
6. スリップ条件：強い思い（誰も知らない場所へ／娘ロザリンドへの愛）、生命の危機（車の事故／刑事との格闘の衝撃）、意識混濁、雷、停電。
7. 過去への反応：「アーチボルト（クラシックカー等の名前で回収）」「ロザリンド（白鳥館の蔵書『お気に召すまま』等で回収）」に無意識に反応。
8. 史実との接続：モーリス・カウリーの事故直後、心身ともに未来で癒やされ、プロットを抱えて1926年12月14日に戻ったため、史実の彼女は無傷でチェックインできている。現代の動画配信（テレビ）も、失踪年の1月にベアード氏のテレビ実験があったため難なく受け入れる。

【ケイマン諸島のトリック概要】

* 舞台：カリブ海のイギリス領、タックス・ヘイヴンにある架空の島「Lesser Cayman」。
* 罠：実在の「オーウェン空港」の名前。Starlinkのアンテナが船と共に去る通信遮断。120V・A型プラグの罠によるWi-Fi電源切れ。非常用のインテルサット（衛星電話）が一般人には使えない絶望。
* アリバイ：現代の乗換案内アプリの「過剰な安全マージン（空白の15分）」を、アガサの「紙の時刻表（ブラッドショー）」のパズル的思考と「高い身体能力（分刻みのダッシュ）」で打ち破る。LCCの個別予約とトランジットエリアでの「闇バイト（身代わり）」を用いた人間の入れ替えトリック。
